% The Abstract Page

\abstract{
\addtocontents{toc}{\vspace{1em}}  % Add a gap in the Contents, for aesthetics

The CMS phase II upgrade outer tracker is built from two types of modules (PS and 2S) each consisting of two silicon sensors and associated electronics and mechanics. In the case of the PS module, the sensors have to be assembled to a precision of approximately 40~um. In order to satisfy this requirement and a short module assembly time, an automated assembly system is proposed. This systems based on a high-precision motion-stage integrated with high-resolution camera, which provides pattern-recognition, vacuum holding system and control systems.

}
